# Security

NZT-48 is experimental play money and is not presented as a financial product,
investment, stable asset, protocol primitive, or audited smart contract.

## Supported Deployments

Only the deployments listed in `deployments/README.md` are canonical:

- Ethereum Mainnet
- Sepolia

The deploy script is kept for local Anvil and throwaway-network experiments.

## Reporting

Do not report security issues through public issues or public pull requests.
Use GitHub's private vulnerability reporting if it is available for this repo,
or contact a maintainer directly.

Please include:

- affected contract address or file path
- a short impact summary
- reproduction steps or a minimal proof of concept

There is no bug bounty program for this repository.
